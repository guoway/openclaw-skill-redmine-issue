# OpenClaw Skill: redmine-issue

> This is an agent skill for [OpenClaw](https://openclaw.ai).

Read Redmine issues via REST API.

## Installation

```bash
clawhub install guoway/openclaw-skill-redmine-issue
```

## Configuration

Set environment variables:
- `REDMINE_URL` - Your Redmine server URL
- `REDMINE_API_KEY` - Your API key
